WebApp em ReactJs -Cadastro de usuários e serviços
<br>
*Feito por Guilherme De Noni Marchioro e Rodrigo Margotti*
<br>
<br>
Front-end -> ReactJs
<br>
Back-end -> JSON Server
<br>
<br>
Comandos para rodar a aplicação:
<br>
1° -> npm install<br>
2° -> json-server --watch data/db.json<br>
3° -> npm run dev<br>
<br>
